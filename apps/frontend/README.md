# Tea Garden — Landing Page

A landing page for a sustainable tea garden / eco brand — hero, gallery, projects, an animated ticker, and a "get involved" call-to-action.

**Stack:** Next.js 16 (App Router) · React 19 · Tailwind CSS 4

## Sections

- `Hero`, `About`, `Gallery`, `Projects`, `Ticker`, `Counter`, `GreenEarthBanner`, `GetInvolved`
- `lib/animations.ts` — shared animation helpers

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
