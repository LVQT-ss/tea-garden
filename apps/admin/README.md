# Admin (placeholder)

Sẽ dựng bằng Next.js (dashboard quản trị: sản phẩm trà, vườn trà, đơn hàng, bài viết...) ở giai đoạn sau, dùng chung API với `apps/backend`.

Khởi tạo gợi ý:

```bash
cd apps/admin
pnpm create next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-pnpm
```
